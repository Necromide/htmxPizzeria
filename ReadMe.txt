Um die Website zum laufen zu bekommen muss bei XAMPP Apache und MySQL gestartet sein.

Der Pfad für den Ordner "Codebasis" ist: "C:\xampp\htdocs\Codebasis"
In dem Ordner befindet sich die Datei "pizzaservice_2020.sql". Diese Datei bei phpMyAdmin importieren, damit die Datenbank existiert.

Nun kann die Seite über einen beliebigen Webbrwoser aufgerufen werden mit folgendem Link: "http://127.0.0.1/codebasis/src/index.php"
